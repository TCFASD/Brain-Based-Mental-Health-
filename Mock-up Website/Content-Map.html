Key Gaps

The biggest gaps across the journey:

  1. Webinars for Steps 2, 3, 4, and 5 are all empty placeholders on the current BBC site. Step 1 has 3 webinars; Steps 2-5 have none.
  2. Step 0 (Why This Matters) needs reflective questions created from scratch and a decision on what tool/handout to offer.
  3. Step 2 (Trauma, FASD, and the Brain) has no handout or tool. The BBC page has empty resource table placeholders.
  4. Self-assessment items need to be broken out by step rather than existing as one 50-item document. The items exist but are not mapped to individual steps in a user-friendly way.
  5. Additional Resources tables on BBC pages for Steps 1, 2, 3, 4, 5 are all empty placeholder tables (Resource Title / Description / Link).
